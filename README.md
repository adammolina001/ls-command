## ls-command

My version os the "ls" terminal's command before starting the shell.

## Compilation
```
  gcc my_ls.c -o my_ls
  ./my_ls          //add options of your choice
```

## Features
  -  ```./my_ls```  show files
  -  ```./my_ls -a```  show files and hidden files (starting with ".")
  -  ```./my_ls -l```  show files names and status
  -  ```./my_ls -la``` or ```./my_ls -al``` to see boths
